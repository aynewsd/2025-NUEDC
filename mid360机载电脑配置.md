# mid360机载电脑配置

## SDK2配置

```bash
mkdir -p ~/livox_ws/src
cd ~/livox_ws/src		#其实随便找个地方放下就行，环境安装了就能用
git clone https://github.com/Livox-SDK/Livox-SDK2.git
cd Livox-SDK2
mkdir build
cd build
cmake .. && make -j4
sudo make install
```

```bash
#测试
cd samples/livox_lidar_quick_start
# WARNING: 
# Remember to modify the mid360_config.json with your PC IP 192.168.1.50
# Run demo1
./livox_lidar_quick_start ../../../samples/livox_lidar_quick_start/mid360_config.json
```

## IP配置

- 机载电脑：

  mid360用网线连接的，其网卡刚需静态IP：

  地址：**192.168.1.50**，子网掩码255.255.255.0，不配置网关（让上网流量走WIFI的网卡）

  机载电脑至少有两个网卡：mid360、地面站局域网，配置方法使用netplan配置

- 虚拟机：

  采用桥接模式连接mid360所在子网，同样设置静态IP

## livox_ros_driver2、FAST_LIO配置

```bash
git clone https://github.com/aynewsd/2025-NUEDC.git
#都在Livox_mid360/，放在包ws_fastlio ws_livox里了

cd ../../../ws_fastlio/

```

## 雷达运行

```bash

source ws_livox/devel/setup.bash
roslaunch livox_ros_driver2 msg_MID360.launch 

#另一个窗口
```
